## Тестовое задание

> Задача: Построить бесконечно расширяемую ленту карточек с сортами кофе

Решение выполнено в `src/`. Функциональный тест находится в `test.ts`.

### Запуск проекта

```shell
pnpm install
pnpm run dev --open
```

### Запуск тестов
```shell
pnpm install
pnpm run test:integration
```
